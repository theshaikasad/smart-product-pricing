How the appraisal works

  1. Encode — the listing text and product photo become 512-d embeddings (OpenCLIP ViT-B/32).
  2. Fuse — a gated cross-attention network with SE residual blocks joins both signals.
  3. Regress — the head predicts log-price; expm1 recovers dollars.

Trained on 75k+ Amazon grocery listings · validation MSAPE 44.4 (single model) / 37.2 (stacked ensemble)
